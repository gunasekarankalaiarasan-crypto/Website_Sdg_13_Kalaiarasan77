Climate Action Hackathon 2025 Website
Project Overview
A responsive website for the Climate Action Hackathon 2025 focused on Urban Heat & Flood Resilience. The site provides event information, registration, and resources for participants.

Files
index.html - Main website structure and content

style.css - Styling and responsive design

image/ - Directory containing all event images

Features
Responsive Design - Works on desktop and mobile devices

Accessibility - Includes skip link for keyboard navigation

Smooth Scrolling - Easy navigation between sections

Registration Form - Participant sign-up with role selection

Speaker Profiles - Showcase of event mentors and speakers

Event Information - Schedule, tracks, and resources
